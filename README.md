# To-do-list
A To-Do List application is a simple program used to manage daily tasks. It helps users to add, view, and remove tasks easily. This project is created using Python and works in a command-line interface. The main aim of this project is to organize tasks in a simple way so that users can keep track of their work. 
# calculator
A Calculator program is a simple application that performs basic arithmetic operations like addition, subtraction, multiplication, and division. The user enters two numbers and selects an operation, and the program displays the result.
# password generator
A Password Generator is a program that creates strong and random passwords based on user input. The user specifies the length of the password, and the program generates a password using a mix of letters, numbers, and special characters.
# Rock - paper-Scissors Game
The Rock-Paper-Scissors game is a simple interactive game between the user and the computer. The user chooses rock, paper, or scissors, and the computer randomly selects one option. The program then compares both choices and decides the winner.
# contact Book
A Contact Book is a program used to store and manage contact details such as name, phone number, and email. The user can add new contacts, view existing contacts, search for a contact, and delete contacts.
